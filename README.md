# Nexus - E-Commerce Platform

A full-stack e-commerce web application built with Node.js, Express.js, and MongoDB following MVC architecture.

**Live Demo:** [nexus-t4sk.onrender.com](https://nexus-t4sk.onrender.com)

## Tech Stack

- **Backend:** Node.js, Express.js, MongoDB, Mongoose
- **Frontend:** EJS, Bootstrap
- **Auth:** express-session, bcryptjs
- **Payment:** MoMo Payment API (UAT)
- **Media:** Cloudinary, Multer
- **Logging:** Winston, Morgan
- **Docs:** Swagger/OpenAPI 3.0
- **Deploy:** Render

## Features

- User authentication & role-based access (Admin/User)
- Product & category management with image upload
- Shopping cart, checkout, and order tracking
- MoMo payment integration with HMAC-SHA256 signature verification
- Flash sale system and product reviews
- Admin dashboard with analytics
- API documentation at `/api-docs`
